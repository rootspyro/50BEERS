## Project Setup

### Requirements

To run this project is required to install:

- Node v22.3.0

### Steps

1 - Enter into the [blog directory](./blog) and install the required node packages.
```shell
$ cd blog
$ npm install
```

2 - Build and Start the project
```shell
$ npm run build
...

$ npm run start
```
